## 카페 단체 주문 시스템

이 프로그램은 카페 단체 주문을 위해 만들어진 시스템입니다.
단체 주문 시 카페 메뉴를 전달하고 각 개인의 주문을 받아 취합하는 번거로움을 줄이기 위해 시스템을 개발했습니다.
